# KidShop – ASP.NET Core 8 MVC
Website bán hàng và LMS được xây dựng bằng ASP.NET Core 8
# Công nghệ sử dụng và thư viện
- ASP.NET Core MVC 8
- Entity Framework Core 8
- SQL Server
- Bootstrap 4
# Chức năng chính
- Quản lý sản phẩm
- Quản lý khoá học
- Giỏ hàng & thanh toán
- Dashboard Admin (CRUD)
- Bình luận & đánh giá
- Thống kê doanh thu
- Tương tác đánh giá
- tích hợp AI chatbot
